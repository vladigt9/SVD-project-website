Randomized Range Approximation Algorithm
    Input: matrix A of size m x n
    Return: Q

    Begin
        Create randomly sampled R (n x l)
        
        O = AR
        Perform QR factorization on O and obtain Q


        For p power iterations
            O' = AQ
            Obtain Q' from: O' = Q'R'
            O'' = AQ'
            Obtain Q from: O'' = QR
        End For
        
        Return Q
    End
